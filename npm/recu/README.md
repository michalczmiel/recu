# recu

CLI tool for tracking recurring expenses. See [github.com/michalczmiel/recu](https://github.com/michalczmiel/recu).

## Install

```
npm install -g @michalczmiel/recu
```

The right prebuilt binary is selected automatically via `optionalDependencies` based on your OS and CPU.

## Supported platforms

- macOS arm64 (`@michalczmiel/recu-darwin-arm64`)

More platforms coming soon.
